# ECommerceDemo
Demonstration on how to create an ecommerce website
This project is an online store meant to show a complete software application. Online shoppers can potentially browse available items, put them into cart for checkout, and be able to confirm and view their selections. The project will show how to use current web tools like HTML, CSS, and JavaScript for what the user sees. It may also utilize Firebase or a different backend to hold shopper info. This project will mainly focus on ease of use and how a normal online store works behind the scenes.
